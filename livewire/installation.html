# Installation

  • Via Composer
  • Create Component

# Via Composer

composer require livewire/livewire

Include the JavaScript (on every page that will be using Livewire).

...
    @livewireStyles
</head>
<body>
    ...
 
    @livewireScripts
</body>
</html>

# Create Component

php artisan make:livewire counter

# Include Component

<head>
    ...
    @livewireStyles
</head>
<body>
    <livewire:counter /> 
 
    ...
 
    @livewireScripts
</body>
</html>
